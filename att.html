HTML (Linguagem de Marcação de HiperTexto) é o bloco de construção mais básico da web. Define o
significado e a estrutura do conteúdo da web. Outras tecnologias do HTML geralmente são
usadas para descrever a aparência/aprensentação (CSS) ou a funcionalidade /comportamento
(JavaScript) de uma página web.

Qual a função do HTML?

A sigla HTML significa Linguagem de Marcação de Hipertexto. Ou seja, o HTML não é considerado uma linguagem de programação
Ele indica onde cada elemento de texto estará estruturando e será mostrado no site ou aplicativo.
O HTML é a linguagem de marcação mais usada no mercado. Antigamente os sites eram estáticos, mas graças ao HTML, eles passaram a ser interativos e muito mais próximos do que são hoje.
Além de ser a linguagem de marcação mais usada, é também considerado padrão da internet, com certificação ISO. Os usos mais comuns de HTML são para o desenvolvimento web, navegação e documentação, muito semelhante ao que se faz no Word.

Tabela de tags de HTML
TAGS UTILIDADE
html Define o início e o fim do documento HTML.
head Define a área do cabeçalho do documento HTML e contém informações sobre o documento, como metadados, título e links para arquivos externos.
body Define a área do corpo do documento HTML, onde todo o conteúdo visível é colocado, como textos, imagens, links, listas, tabelas, formulários, entre outros elementos.
div Cria uma caixa de conteúdo que pode ser estilizada com CSS e usada para agrupar outros elementos HTML.
p Cria um parágrafo de texto.
h1 - h6 Cria títulos e subtítulos de diferentes tamanhos e níveis de importância.
img Insere uma imagem no documento HTML, especificando a URL da imagem, o texto alternativo para fins de acessibilidade e outras informações opcionais, como tamanho, alinhamento e borda.
sub formata um texto subscrito
b formata um texo em negrito.
ol Cria uma lista ordenada, onde os itens da lista são numerados em ordem sequencial.
  1. IP - Internet Protocol
  2. SSH - Secure Shell
  3. FTP - File Transfer Protocol
  4. POP3 - Post Office Protocol 3
  5. SFTP - Simple Transfer Protocol

O que é CSS

CSS é chamado de linguagem Cascading Style Sheet e é usado para estilizar elementos escritos em uma linguagem de marcação como HTML. O CSS separa o conteúdo da representação visual do site. Pense na decoração da sua página. Utilizando o CSS é possível alterar a cor do texto e do fundo, fonte e espaçamento entre parágrafos. Também pode criar tabelas, usar variações de layouts, ajustar imagens para suas respectivas telas e assim por diante.

O que é HTML

A Linguagem de Marcação de Hipertexto (HTML) é uma linguagem de computador que compõe a maior parte das páginas da internet e dos aplicativos online. Um hipertexto é um texto usado para fazer referência a outros textos, enquanto uma linguagem de marcação é composta por uma série de marcações que dizem para os servidores da web qual é o estilo e a estrutura de um documento.

O que é JavaScript

O JS ou JavaScript é uma linguagem de programação de alto-nível, criada no meio da década de 90, mais precisamente em 1996 pelo lendário programador Brendan Eich que, além de criar o JavaScript, foi também um dos fundadores da Mozilla Corporation.
